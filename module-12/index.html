Experience

My personal experience includes a little under 2 years of personal practice with C# using a project targeted for production. Since I am not in a position to take on an internship, I have decided to use a real-world project to demonstrate my abilities. I am building a project with Visual Studio for Mac using .NET Core. The project consists of a Web API; MAUI cross-platform app; Contracts Class Library; Unit Test project; and Integration Test project. The project utilizes a MySQL Docker image for a development database; Serilog for event logging; and Seq as a dashboard for filtering and viewing those event logs. The project is version-controlled using Git and GitHub. The project uses token-based authentication with Json Web Tokens. The project is being developed using the Test-Driven Development framework in a Waterfall-style process (since the project is small enough that Agile isn't needed).
