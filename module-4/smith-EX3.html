5 Things I've Learned Since the Course Started

  • Outside of XTHML Strict, empty tags don't need a forward slash.
  • A lot of good programming practices, like code structure and good documentation, are applicable to HTML and CSS.
  • Indenting tags and spacing blocks similarly to program code makes it much easier to read.
  • I like Visual Studio Code more than Notepad++ for editing HTML and CSS files.
  • GitHub can display HTML files in a repository as a separate site.
